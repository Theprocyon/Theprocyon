# Theprocyon

하드웨어, 펌웨어 엔지니어

### information
- 인하대학교 정보통신공학과 학부생(휴학)
- 스타트업에서 정보처리분야 산업기능요원으로 재직중

### HW
- Power Electronics (Power conversion, Battery, Motor Control)
- RF (GPS, Wifi)
- MCU + Pheriphral HW (STM32, AVR, nRF)
- USB / Etherent HW
- EMC (EMI Debugging)
- PCB Artwork

### FW
- C / C++
- STM32
- nRF52 Series

### Tools
- SW : IDEs, Altium Designer, Autodesk Inventor, Ansys HFSS
- HW : Oscilloscope, Function Generator, Spectrum Analyzer, Vector Network Analyzer
